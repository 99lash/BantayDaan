# Build with AI: Mini-Hackathon Details

**Theme:** Orchestrate the Future. Ship the Agent.

---

## 🏆 Evaluation Criteria

| Criteria | Weight | What we're looking for |
| :--- | :--- | :--- |
| **Practical Solution** | 50% | Does it solve a real problem effectively? |
| **Technical Depth** | 20% | Correct integration of Gemini API and Agentic logic. |
| **Aesthetics/UI** | 20% | Polished, consistent design and clear layout. |
| **User Experience** | 10% | Intuitive flow, "Time-to-Value" for the user. |

---

## 📂 Project Categories

### Category 1: The Agentic Architect (Tech Focused)
*   **Focus:** Tool calling, task planning, and multi-step reasoning.
*   **Requirement:** Demonstrate the agent interacting with an external API or data structure.
*   **Goal:** Build AI agents that perform tasks autonomously. Your system shouldn't just talk; it should do.

### Category 2: Local Impact Explorer (Community Focused)
*   **Focus:** Hyper-local context and specific user pain points.
*   **Requirement:** A clear problem statement explaining why "Global AI" fails here.
*   **Goal:** Solve a problem unique to the Philippines or Southeast Asia (e.g., "Paano maiiwasan ang traffic sa EDSA" logic).

### Category 3: The Chaos-to-Code Converter (Wildcard)
*   **Focus:** Creativity, humor, and automation of "chaos."
*   **Requirement:** Must automate at least a 3-step workflow using AI decision-making.
*   **Goal:** Automate the "unnecessary." Turn a hilarious daily annoyance or a random Filipino meme into a functional, automated workflow.
